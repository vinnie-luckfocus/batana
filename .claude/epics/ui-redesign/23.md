# Task: 分析进度可视化

## Description

重构分析进度界面（AnalysisProgressScreen），解决当前进度不可见、用户焦虑的问题。提供大号圆形进度条、分阶段说明、预计剩余时间和取消操作，让用户清楚了解分析状态。

## Acceptance Criteria

- [ ] 采用设计系统中的 Neumorphic 组件
- [ ] 大号圆形进度条（Neumorphic 风格）
- [ ] 分阶段说明：姿态识别 → 指标计算 → 评分生成
- [ ] 实时进度百分比显示
- [ ] 预计剩余时间显示
- [ ] 取消按钮（带二次确认对话框）
- [ ] 分析完成后自动跳转到结果展示页面
- [ ] 分析失败时显示友好错误提示

## Technical Details

### 布局结构
```
AnalysisProgressScreen
├── AppBar (Neumorphic)
│   └── 标题："分析中"
├── Body (Center)
│   ├── CircularProgressIndicator (大号 Neumorphic)
│   │   └── 进度百分比（中心显示）
│   ├── StageIndicator
│   │   ├── 姿态识别 (0-33%)
│   │   ├── 指标计算 (34-66%)
│   │   └── 评分生成 (67-100%)
│   ├── TimeEstimate
│   │   └── "预计剩余 8 秒"
│   └── CancelButton (Neumorphic)
└── 二次确认对话框
```

### 状态管理
- 使用 Provider 管理分析进度状态
- `AnalysisProgressState`: 管理进度百分比、当前阶段、剩余时间

### 进度计算
```dart
// 根据 MediaPipe 分析进度更新 UI
enum AnalysisStage {
  poseDetection,    // 0-33%
  metricCalculation, // 34-66%
  scoreGeneration,   // 67-100%
}
```
