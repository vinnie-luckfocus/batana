# Task: 历史记录界面优化

## Description

重构历史记录界面（HistoryScreen），解决当前列表单调、对比困难的问题。采用时间线布局，支持筛选、排序和快速对比功能，提供滑动操作（删除、对比）。

## Acceptance Criteria

- [ ] 采用设计系统中的 Neumorphic 组件
- [ ] 时间线布局，按日期分组
- [ ] 卡片展示：缩略图 + 评分 + 日期
- [ ] 筛选功能：按日期范围、评分区间
- [ ] 排序功能：按日期、按评分
- [ ] 快速对比功能：选择 2-3 条记录对比
- [ ] 滑动操作：左滑删除、右滑对比
- [ ] 支持下拉刷新
- [ ] 懒加载（分页加载）

## Technical Details

### 布局结构
```
HistoryScreen
├── AppBar (Neumorphic)
│   ├── 标题："历史记录"
│   └── 筛选/排序按钮
├── FilterBar (可展开)
│   ├── 日期范围选择
│   └── 评分区间选择
├── Body (ListView)
│   ├── DateSection: "今天"
│   │   ├── HistoryCard 1
│   │   └── HistoryCard 2
│   ├── DateSection: "昨天"
│   │   └── HistoryCard 3
│   └── DateSection: "本周"
│       └── HistoryCard 4
└── FloatingActionButton
    └── 对比按钮（选中 ≥ 2 条时显示）
```

### 状态管理
- 使用 Provider 管理历史记录状态
- `HistoryState`: 管理记录列表、筛选条件、选中项

### 滑动操作
```dart
Dismissible(
  key: Key(record.id),
  direction: DismissDirection.horizontal,
  background: DeleteBackground(),  // 左滑删除
  secondaryBackground: CompareBackground(),  // 右滑对比
  onDismissed: (direction) {
    if (direction == DismissDirection.startToEnd) {
      _deleteRecord(record);
    } else {
      _addToCompare(record);
    }
  },
)
```

### 分页加载
- 每页加载 20 条记录
- 滚动到底部自动加载下一页
