# Task: 结果展示界面重构

## Description

重构结果展示界面（ResultScreen），解决当前信息层级不清晰的问题。优化综合评分展示，提供分项评分卡片、关键帧轮播和改进建议，支持分享和保存功能。

## Acceptance Criteria

- [ ] 采用设计系统中的 Neumorphic 组件
- [ ] 顶部：综合评分（大号显示，视觉突出）
- [ ] 中部：分项评分卡片（可横向滑动）
  - 挥棒速度
  - 身体协调
  - 击球点
  - 其他维度
- [ ] 底部：关键帧轮播 + 改进建议
- [ ] 操作按钮：分享、保存、重新分析
- [ ] 支持下拉刷新
- [ ] 适配不同屏幕尺寸

## Technical Details

### 布局结构
```
ResultScreen
├── AppBar (Neumorphic)
│   └── 标题："分析结果"
├── Body (ScrollView)
│   ├── OverallScoreCard (大号 Neumorphic)
│   │   ├── 综合评分（80 分）
│   │   └── 评级（优秀/良好/一般）
│   ├── DetailScoresSection
│   │   └── HorizontalScrollView
│   │       ├── ScoreCard: 挥棒速度 (85)
│   │       ├── ScoreCard: 身体协调 (78)
│   │       ├── ScoreCard: 击球点 (82)
│   │       └── ScoreCard: 其他维度
│   ├── KeyFramesCarousel
│   │   └── PageView: 关键帧图片
│   └── SuggestionsSection
│       └── 改进建议列表
└── BottomActions
    ├── ShareButton
    ├── SaveButton
    └── RetryButton
```

### 状态管理
- 使用 Provider 管理结果展示状态
- `ResultState`: 管理分析结果数据、分享状态

### 数据模型
```dart
class AnalysisResult {
  final int overallScore;
  final Map<String, int> detailScores;
  final List<String> keyFrames;
  final List<String> suggestions;
}
```
