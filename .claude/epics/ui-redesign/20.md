# Task: 主界面重构

## Description

重新设计主界面（HomeScreen），解决当前布局混乱、功能入口不清晰的问题。采用卡片式布局，提供清晰的三大功能入口（录制视频、选择相册、历史记录），支持快速访问最近分析结果，使用底部导航栏实现主要页面切换。

**详细设计文档**: [ui-design-v2.md](./design/ui-design-v2.md)

## Acceptance Criteria

- [ ] 采用设计系统中的 Neumorphic 组件
- [ ] 实现卡片式布局，清晰分区
- [ ] 三大功能入口：录制视频、选择相册、历史记录
- [ ] 快速访问区域：显示最近 3 次分析结果
- [ ] 底部导航栏：主页、历史、设置
- [ ] 页面响应时间 < 100ms
- [ ] 支持下拉刷新
- [ ] 适配不同屏幕尺寸（iPhone SE 到 iPad）

## Technical Details

### 布局结构
```
HomeScreen
├── AppBar (Neumorphic)
│   └── 标题 + 用户头像（占位）
├── Body (ScrollView)
│   ├── 功能卡片区
│   │   ├── 录制视频卡片
│   │   ├── 选择相册卡片
│   │   └── 历史记录卡片
│   └── 最近分析区
│       └── 最近 3 次分析结果列表
└── BottomNavigationBar (Neumorphic)
    ├── 主页
    ├── 历史
    └── 设置
```

### 状态管理
- 使用 Provider 管理主界面状态
- `HomeState`: 管理最近分析结果、加载状态

### 路由配置
```dart
Navigator.pushNamed(context, '/record');  // 录制视频
Navigator.pushNamed(context, '/gallery'); // 选择相册
Navigator.pushNamed(context, '/history'); // 历史记录
```
