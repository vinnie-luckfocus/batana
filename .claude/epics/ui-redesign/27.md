# Task: 设计系统基础建设

## Description

建立独立的设计系统模块（Design System），为整个 UI 重新设计提供统一的视觉规范和组件基础。集成 Flutter-Neumorphic 库，定义色彩系统、字体系统、间距系统和 Neumorphic 主题配置，开发基础组件库（按钮、卡片、进度指示器等）。

这是整个 UI 重新设计的基础任务，所有其他页面重构任务都依赖于此任务的完成。

## Acceptance Criteria

- [ ] 创建 `lib/design_system/` 目录结构
- [ ] 集成 `flutter_neumorphic` 包（版本 ^3.2.0）
- [ ] 定义色彩系统（主色、辅助色、强调色）
- [ ] 定义字体系统（标题、正文、辅助文字）
- [ ] 定义间距系统（统一的 padding 和 margin 规范）
- [ ] 配置 Neumorphic 主题（阴影深度、模糊、偏移）
- [ ] 开发基础组件：NeumorphicButton、NeumorphicCard、NeumorphicProgressIndicator
- [ ] 创建组件示例页面（Storybook）用于预览和测试
- [ ] 编写设计系统使用文档

## Technical Details

### 目录结构
```
lib/design_system/
├── colors.dart          # 色彩系统定义
├── typography.dart      # 字体系统定义
├── spacing.dart         # 间距系统定义
├── neumorphic_theme.dart # Neumorphic 主题配置
└── widgets/             # 通用组件库
    ├── buttons.dart     # 按钮组件
    ├── cards.dart       # 卡片组件
    └── progress_indicators.dart # 进度指示器
```

### 色彩系统
- 主色调：科技蓝（#2196F3）
- 辅助色：温暖灰（#ECEFF1）
- 强调色：活力橙（#FF9800）
- 背景色：浅灰（#F5F5F5）
- 文字色：深灰（#212121）

### Neumorphic 配置
- 阴影深度：8-16
- 模糊半径：16-24
- 光源方向：左上（-5, -5）

### 基础组件
1. **NeumorphicButton**: 支持不同尺寸（small, medium, large）和状态（normal, pressed, disabled）
2. **NeumorphicCard**: 支持不同圆角和阴影深度
3. **NeumorphicProgressIndicator**: 圆形和线性进度条

### 依赖包
```yaml
dependencies:
  flutter_neumorphic: ^3.2.0
```
