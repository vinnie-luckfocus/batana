# Task: 设计系统基础建设

## Description

建立独立的设计系统模块（Design System），为整个 UI 重新设计提供统一的视觉规范和组件基础。集成 Flutter-Neumorphic 库，定义色彩系统、字体系统、间距系统和 Neumorphic 主题配置，开发基础组件库（按钮、卡片、进度指示器等）。

这是整个 UI 重新设计的基础任务，所有其他页面重构任务都依赖于此任务的完成。

### 设计原则

**审美高级**
- 采用现代 Neumorphic 设计语言，营造柔和、立体的视觉效果
- 色彩搭配遵循 60-30-10 原则（主色 60%、辅助色 30%、强调色 10%）
- 使用微妙的渐变和阴影，避免过度扁平或过度拟物
- 保持视觉层次清晰，重要元素突出，次要元素收敛

**界面布局合理**
- 遵循 8px 网格系统，确保元素对齐和间距一致
- 采用黄金比例（1.618）设计组件尺寸关系
- 信息密度适中，避免拥挤或空旷
- 响应式布局，适配不同屏幕尺寸（手机、平板）

**交互友好**
- 所有交互元素提供清晰的视觉反馈（按压、悬停、禁用状态）
- 动画过渡自然流畅（推荐 200-300ms 缓动曲线）
- 触摸目标尺寸 ≥ 44x44 pt（符合 iOS/Android 人机交互规范）
- 错误状态友好提示，成功状态给予正向反馈

## Technical Details

### 目录结构
```
lib/design_system/
├── colors.dart          # 色彩系统定义
├── typography.dart      # 字体系统定义
├── spacing.dart         # 间距系统定义（8px 网格）
├── radius.dart          # 圆角规范
├── animations.dart      # 动画配置（时长、曲线）
├── neumorphic_theme.dart # Neumorphic 主题配置
└── widgets/             # 通用组件库
    ├── buttons.dart     # 按钮组件
    ├── cards.dart       # 卡片组件
    ├── text_fields.dart # 输入框组件
    ├── switches.dart    # 开关组件
    └── progress_indicators.dart # 进度指示器
```

### 色彩系统（高级审美）
**主色调（60%）**
- Primary: `#1E88E5`（深邃蓝，专业感）
- Primary Light: `#64B5F6`（浅蓝，辅助）
- Primary Dark: `#1565C0`（深蓝，强调）

**辅助色（30%）**
- Surface: `#ECEFF1`（温暖灰，卡片背景）
- Background: `#F5F7FA`（浅灰蓝，页面背景）
- Divider: `#CFD8DC`（分割线）

**强调色（10%）**
- Accent: `#FF6F00`（活力橙，CTA 按钮）
- Success: `#43A047`（成功绿）
- Warning: `#FFA726`（警告橙）
- Error: `#E53935`（错误红）

**文字色（层次分明）**
- Text Primary: `#212121`（主要文字，87% 不透明度）
- Text Secondary: `#757575`（次要文字，60% 不透明度）
- Text Disabled: `#BDBDBD`（禁用文字，38% 不透明度）

### 字体系统（清晰层次）
```dart
// 标题
Display: 32pt / Bold / 1.2 行高
H1: 24pt / Bold / 1.3 行高
H2: 20pt / SemiBold / 1.4 行高
H3: 18pt / Medium / 1.4 行高

// 正文
Body Large: 16pt / Regular / 1.5 行高
Body: 14pt / Regular / 1.5 行高
Body Small: 12pt / Regular / 1.5 行高

// 辅助
Caption: 12pt / Regular / 1.4 行高
Overline: 10pt / Medium / 1.4 行高（全大写）
```

### 间距系统（8px 网格）
```dart
// 基础间距
XXS: 4pt   // 极小间距（图标与文字）
XS: 8pt    // 小间距（组件内部）
S: 12pt    // 中小间距（相关元素）
M: 16pt    // 标准间距（默认）
L: 24pt    // 大间距（组件间）
XL: 32pt   // 超大间距（区块间）
XXL: 48pt  // 页面级间距
```

### 圆角规范（统一视觉）
```dart
Small: 8pt   // 按钮、标签
Medium: 12pt // 卡片、输入框
Large: 16pt  // 大卡片、弹窗
XLarge: 24pt // 特殊容器
```

### Neumorphic 配置（柔和立体）
```dart
// 阴影配置
Light Shadow: Color(0x1A000000), offset: (-4, -4), blur: 8
Dark Shadow: Color(0x33000000), offset: (4, 4), blur: 8

// 深度级别
Depth 1: 浅浮起（按钮默认）
Depth 2: 中等浮起（卡片）
Depth 3: 深度浮起（弹窗）
Depth -1: 凹陷（输入框、开关）

// 曲率
Flat: 0.0      // 完全平面
Concave: 0.5   // 轻微凹陷
Convex: 1.0    // 轻微凸起
```

### 动画规范（流畅自然）
```dart
// 时长
Fast: 150ms    // 快速反馈（按钮按压）
Normal: 250ms  // 标准过渡（页面切换）
Slow: 400ms    // 慢速动画（复杂变化）

// 缓动曲线
EaseOut: Curves.easeOut        // 元素进入
EaseIn: Curves.easeIn          // 元素退出
EaseInOut: Curves.easeInOut    // 状态切换
Spring: Curves.elasticOut      // 弹性效果
```

### 基础组件（交互友好）

#### 1. NeumorphicButton
**尺寸规范**
- Small: 32pt 高度，12pt 内边距
- Medium: 44pt 高度，16pt 内边距（默认）
- Large: 56pt 高度，24pt 内边距

**状态设计**
- Normal: 默认阴影，Primary 色
- Hover: 阴影加深 10%（Web/Desktop）
- Pressed: 凹陷效果，阴影反转，触觉反馈
- Disabled: 50% 不透明度，无阴影

**样式变体**
- Filled: 实心按钮（主要操作）
- Outlined: 描边按钮（次要操作）

#### 2. NeumorphicCard
**布局规范**
- 内边距：16pt（标准）/ 24pt（宽松）
- 圆角：12pt（默认）
- 阴影深度：Depth 2

**内容布局**
- 标题 + 内容 + 操作区（垂直排列）
- 支持头部图片（16:9 比例）
- 底部操作按钮右对齐

#### 3. NeumorphicProgressIndicator
**圆形进度条**
- 直径：48pt / 64pt / 80pt
- 线宽：4pt / 6pt / 8pt
- 动画：顺时针旋转，1.5s 周期

**线性进度条**
- 高度：4pt / 6pt
- 圆角：2pt / 3pt
- 动画：从左到右填充，缓动曲线

#### 4. NeumorphicTextField
**尺寸规范**
- 高度：48pt（符合触摸目标）
- 内边距：12pt 水平，12pt 垂直
- 圆角：12pt

**状态设计**
- Normal: 凹陷效果，浅灰背景
- Focused: 边框高亮（Primary 色），阴影加深
- Error: 红色边框，抖动动画，错误提示
- Disabled: 50% 不透明度，无交互

#### 5. NeumorphicSwitch
**尺寸规范**
- 宽度：52pt，高度：32pt
- 滑块直径：28pt
- 触摸区域：≥ 44x44pt

**动画设计**
- 滑块移动：250ms，EaseInOut 曲线
- 背景色渐变：250ms
- 开启时触觉反馈（轻微震动）
