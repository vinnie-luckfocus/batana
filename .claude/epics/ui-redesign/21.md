# Task: 录制界面优化

## Description

优化录制界面（RecordScreen），解决当前控件重叠、操作不直观的问题。采用全屏相机预览，使用浮动控制按钮，提供清晰的录制状态反馈和操作指引。

## Acceptance Criteria

- [ ] 采用设计系统中的 Neumorphic 组件
- [ ] 全屏相机预览（无黑边）
- [ ] 浮动控制按钮：开始/停止/重录
- [ ] 顶部状态栏：录制时长、网格辅助线开关
- [ ] 底部提示：录制指引文字
- [ ] 录制时长实时显示
- [ ] 网格辅助线可选（九宫格）
- [ ] 录制完成后自动跳转到分析进度页面
- [ ] 支持横竖屏切换

## Technical Details

### 布局结构
```
RecordScreen
├── CameraPreview (全屏)
├── TopBar (半透明)
│   ├── 返回按钮
│   ├── 录制时长
│   └── 网格开关
├── FloatingControls (底部居中)
│   ├── 重录按钮
│   ├── 录制/停止按钮（大号）
│   └── 占位（对称布局）
└── BottomHint
    └── 提示文字（"保持设备稳定，录制 5-10 秒"）
```

### 状态管理
- 使用 Provider 管理录制状态
- `RecordState`: 管理录制状态（idle, recording, paused）、录制时长、网格显示

### 相机配置
- 分辨率：1080p
- 帧率：30fps
- 方向：自动适配

### 文件位置
- `lib/screens/record/record_screen.dart`
- `lib/screens/record/widgets/camera_preview_widget.dart`
- `lib/screens/record/widgets/recording_controls.dart`
- `lib/providers/record_state.dart`
