# Task: 设置界面与性能优化

## Description

重构设置界面（SettingsScreen），提供清晰的分组设置选项。进行全局性能优化，确保界面响应时间 < 100ms，页面切换动画流畅（60fps），完成全面测试和 Bug 修复。

## Acceptance Criteria

- [ ] 采用设计系统中的 Neumorphic 组件
- [ ] 分组设置：通用、权限、存储、关于
- [ ] 通用设置：网格辅助线默认开关、深色模式（占位）
- [ ] 权限管理：相机权限、相册权限状态显示和引导
- [ ] 存储管理：显示占用空间、清理缓存功能
- [ ] 关于页面：版本信息、帮助文档、隐私政策
- [ ] 全局性能优化：界面响应 < 100ms，动画 60fps
- [ ] 内存优化：正常使用 < 200MB
- [ ] 全面测试：单元测试、Widget 测试、集成测试
- [ ] Bug 修复：解决所有已知问题

## Technical Details

### 布局结构
```
SettingsScreen
├── AppBar (Neumorphic)
│   └── 标题："设置"
└── Body (ListView)
    ├── SettingsSection: "通用"
    │   ├── SwitchTile: 网格辅助线
    │   └── SwitchTile: 深色模式（占位）
    ├── SettingsSection: "权限"
    │   ├── PermissionTile: 相机权限
    │   └── PermissionTile: 相册权限
    ├── SettingsSection: "存储"
    │   ├── StorageInfo: 已用空间
    │   └── ActionTile: 清理缓存
    └── SettingsSection: "关于"
        ├── InfoTile: 版本号
        ├── ActionTile: 帮助文档
        └── ActionTile: 隐私政策
```

### 性能优化清单

- [ ] 使用 `const` 构造函数减少重建

#### 2. 动画优化
