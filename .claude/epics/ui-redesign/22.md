# Task: 相册视频选择功能

## Description

实现从相册选择视频进行分析的新功能。集成 image_picker 插件，支持从系统相册选择视频，提供视频预览、格式验证和时长检查，选择完成后自动跳转到分析进度页面。

## Acceptance Criteria

- [ ] 集成 image_picker 插件（版本 ^1.0.0）
- [ ] 支持从相册选择视频
- [ ] 支持视频预览（缩略图 + 播放）
- [ ] 显示视频时长和分辨率
- [ ] 支持常见格式：MP4, MOV, AVI
- [ ] 格式验证：不支持的格式显示友好提示
- [ ] 时长验证：建议 5-30 秒，超出范围给出警告
- [ ] 选择完成后跳转到分析进度页面
- [ ] iOS 和 Android 权限配置正确

## Technical Details

### 权限配置

**iOS (Info.plist)**:
```xml
<key>NSPhotoLibraryUsageDescription</key>
<string>需要访问相册以选择训练视频进行分析</string>
```

**Android (AndroidManifest.xml)**:
```xml
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"/>
```

### 实现方案
```dart
// 使用 image_picker 选择视频
final XFile? video = await ImagePicker().pickVideo(
  source: ImageSource.gallery,
  maxDuration: Duration(seconds: 30),
);

// 验证视频格式和时长
if (video != null) {
  final isValid = await _validateVideo(video);
  if (isValid) {
    Navigator.pushNamed(context, '/analysis', arguments: video.path);
  }
}
```

### 状态管理
- 使用 Provider 管理视频选择状态
- `VideoSourceState`: 管理视频来源（录制/相册）、视频路径

### 文件位置
