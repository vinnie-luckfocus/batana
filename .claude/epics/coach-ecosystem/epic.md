# Epic: coach-ecosystem

## Overview

构建训练生态闭环：教练模式（多学员管理）→ AI 训练计划生成 → 每日任务系统 → 周期进度报告 → 成就激励 → 教练-学员协作。目标是从"分析工具"升级为"训练平台"，为教练提供批量管理与数据驱动决策能力，为学员提供结构化训练路径与持续激励，实现 8-12 周训练周期内的可观改善。

## Architecture Decisions

- **教练模式**：独立的教练账号体系，支持多学员管理与分组
- **AI 训练计划**：基于学员历史数据分析弱项，生成个性化训练计划
- **任务系统**：每日任务推送、完成打卡、进度追踪
- **成就系统**：徽章、里程碑、排行榜（可选）
- **协作工具**：消息通知、视频批注、计划同步

## Technical Approach

### 客户端模块
- **coach_mode**：教练账号、学员管理、分组、批量操作
- **training_plan**：AI 计划生成、计划模板库、手动编辑
- **task_system**：每日任务、完成打卡、进度统计
- **achievement**：成就系统、里程碑、排行榜
- **collaboration**：消息通知、视频批注、计划同步

### 后端服务
- 教练-学员关系管理 API
- AI 训练计划生成服务
- 周期报告生成 API
- 推送通知服务

### AI/ML 层
- 学员弱项分析算法
- 训练计划生成模型
- 进步预测模型

## Implementation Strategy

### 开发阶段
- M1（4 周）：教练模式基础（账号 + 学员管理 + 分组）
- M2（4 周）：AI 训练计划生成 + 计划模板库
- M3（3 周）：每日任务系统 + 成就系统
- M4（3 周）：周期报告 + 教练批注 + 消息通知
- M5（2 周）：教练/学员测试 + 反馈迭代

### 风险缓解
- AI 训练计划建立人工审核与调整机制
- 提供教练免费试用与培训
- A/B 测试优化激励机制
