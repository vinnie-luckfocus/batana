# Task: 006 - 规则评分引擎 + 建议模板库

## Description
实现规则评分引擎、问题诊断映射、可执行建议生成

## Acceptance Criteria
- [ ] 综合评分计算 (0-100, 加权计算)
- [ ] 问题模式识别 (过早开肩、重心后移不足等)
- [ ] 建议模板库 (至少 10 条建议)
- [ ] 建议使用通俗解释 + 可执行动作格式

## Technical Details
- **综合评分计算**：
  - 速度指标权重：35%
  - 角度指标权重：35%
  - 协调性指标权重：30%
  - 加权计算总分 0-100
- **问题模式识别**：
  - 过早开肩：肩转动超前于髋转动超过阈值
  - 重心后移不足：击球时重心仍在后方
  - 挥棒速度不足：速度低于合理范围
  - 挥棒角度过大/过小：角度偏离理想范围
  - 协调性不足：髋肩时序差过大
- **建议模板库**：
  - 至少 10 条建议
  - 每条建议包含：通俗解释 + 可执行动作
  - 示例："挥棒时保持髋部领先" → "练习击球练习时，注意先转髋再转肩"
