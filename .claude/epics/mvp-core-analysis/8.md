# Task: 007 - 分析结果页 + 历史记录页 + SQLite 存储

## Description
实现结果展示 UI、历史记录列表、SQLite 本地存储

## Acceptance Criteria
- [ ] 结果页: 综合评分(进度环)、3个核心指标、改进建议
- [ ] 历史页: 最近10条记录、时间倒序、点击查看详情
- [ ] SQLite: 记录存储、自动清理最旧记录

## Technical Details
- **结果页 UI**：
  - 综合评分：进度环组件显示 0-100 分
  - 3个核心指标：速度、角度、协调性（带数值和参考标签）
  - 改进建议：列表展示建议（通俗解释 + 可执行动作）
- **历史记录页 UI**：
  - 显示最近 10 条记录
  - 时间倒序排列（最新的在前）
  - 点击记录可查看详情
  - 包含评分和日期
- **SQLite 存储**：
  - 分析记录表：id, 综合评分, 速度, 角度, 协调性, 建议(JSON), 视频路径, 创建时间
  - 自动清理：保留最近 50 条记录，自动删除最旧记录
  - 数据持久化：应用重启后数据保留
