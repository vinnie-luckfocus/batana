# Task: 004 - 挥棒阶段分割算法

## Description
基于腕部速度曲线识别准备/加速/击球/收尾四阶段，支持降级到三阶段

## Acceptance Criteria
- [ ] 腕部关键点速度曲线计算
- [ ] 四阶段分割 (准备期、加速期、击球期、收尾期)
- [ ] 三阶段降级策略 (启动-击球-收尾)
- [ ] 分割失败的用户提示

## Technical Details
- 基于腕部关键点 (WRIST_RIGHT) 计算速度曲线
- 实现四阶段分割算法：
  - 准备期：挥棒前静止阶段
  - 加速期：腕部开始加速到最大速度
  - 击球期：最大速度点到击球点
  - 收尾期：击球后减速到静止
- 实现三阶段降级策略（当四阶段分割失败时）
- 提供清晰的分割结果和用户提示
