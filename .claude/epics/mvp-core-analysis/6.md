# Task: 005 - 核心指标计算（速度/角度/协调性）

## Description
计算挥棒速度、挥棒角度、动作协调性三个核心指标

## Acceptance Criteria
- [ ] 速度计算 (基于身高标定的相对速度)
- [ ] 角度计算 (挥棒平面与水平面夹角)
- [ ] 协调性计算 (髋肩启动时序差、重心转移流畅度)
- [ ] 指标结果标注"参考值"

## Technical Details
- **速度计算**：
  - 基于腕部关键点轨迹计算绝对速度
  - 使用身高进行标定，转换为相对速度 (m/s)
  - 标注为"参考值"表明为估算值
- **角度计算**：
  - 计算挥棒平面（腕部-肩部连线）与水平面的夹角
  - 识别最大挥棒角度
  - 标注为"参考值"
- **协调性计算**：
  - 髋肩启动时序差：髋部转动与肩部转动的时间差
  - 重心转移流畅度：基于重心关键点轨迹的平滑度评估
  - 标注为"参考值"
